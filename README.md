# stsnp
STSNP is a CLI(shortly GUI) tool that simplifies launching Snap applications on minimal window managers like Openbox.

## Why?
Using Snap apps in minimal environments can be inconvenient.<br>
STSNP provides a faster and cleaner way to manage and run them.

## Features
- Run Snap apps quickly
- List avaiable apps
- Ignor unwanted apps (By using IGNOR list)

## Usage

Run application:<br>
`python3 main.py run firefox`

Show availble apps:<br>
`python3 main.py show`

## Requirements
- Linux
- Python3
- Snap installed
