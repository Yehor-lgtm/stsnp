# stsnp
STSNP is a tool for launching snap applications on Linux systems, especially designed for minimal wm like Openbox. <br>The application accepts arguments and allows you not to use them, so far only CLI, but GUI is planned.<br>
<br>
You can delete or add apps to IGNOR, if you want<br>
USAGE: python3 main.py (command) (app)<br>
<br>
Commands: run, show
